# 🌪️ AirFlow AI Pro: Invisible Wind Visualization

Transform subtle movements into visible airflow paths using AI-driven Dense Optical Flow and Particle Physics.

## 🚀 Features
- **Turbulence Detection:** Real-time vorticity mapping to find swirling air.
- **Breathing Airflow:** Ultra-sensitive mode for physiological monitoring.
- **Thermal Mapping:** Visualize air speed as a heat signature.
- **Airflow Prediction:** Vector-based trajectory forecasting.
- **Atmospheric HUD:** Professional interface with live intensity and chaos metrics.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **Engine:** OpenCV (Dense Optical Flow)
- **Math:** Numpy

## 📦 Installation
1. Clone the repo:
   ```bash
   git clone <your-repo-url>
   cd AirFlow_AI_Project
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   python -m streamlit run app.py
   ```

## 🔒 Security
- No API keys required.
- No data is uploaded to the cloud (Local-first).
